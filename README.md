# event Registration Form

This project is a simple HTML form for user event registration.

## Features
- Input validation (name, email, phone)
- Event selection dropdown
- Client-side validation using JavaScript

## Technologies Used
- HTML
- CSS (optional)
- JavaScript

## How to Use
1. Clone this repository.
2. Open the `event_registration.html` file in your browser.
3. Fill the form and submit to test validation.
